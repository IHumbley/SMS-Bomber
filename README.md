**A Simple Iranian Sms bomber**

# Requirements
-   Python _v3.x.x_
-   Proxy/Tor

# Run
1. git clone https://github.com/MrSubmissive/SMS-Bomber
2. python3 sms_bomber.py

# Tip
If You want to use this script with HTTP/HTTPS proxy you can run sms_bomberProxyVersion.py
